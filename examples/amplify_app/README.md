# examples/amplify_app

Deploy **amplify_app** resources via Terraform.

---

## Quick start
```bash
just plan amplify_app
just apply amplify_app
```

## Show outputs
```bash
just output amplify_app
```

## Clean up
```bash
just destroy-apply amplify_app
```
