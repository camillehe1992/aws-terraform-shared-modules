# examples/ecs_cluster

Deploy **ecs_cluster** resources via Terraform.

---

## Quick start
```bash
just plan ecs_cluster
just apply ecs_cluster
```

## Show outputs
```bash
just output ecs_cluster
```

## Clean up
```bash
just destroy-apply ecs_cluster
```
