# examples/s3_static_website

Deploy **s3_static_website** resources via Terraform.

---

## Quick start
```bash
just plan s3_static_website
just apply s3_static_website
```

## Show outputs
```bash
just output s3_static_website
```

## Clean up
```bash
just destroy-apply s3_static_website
```
