# Setup a REST API based on AWS API Gateway and Lambda

The example is used to demostrate a serverless REST API application based on AWS API Gateway and Lambda.

---

## Quick start

```bash
just plan _template
just apply _template
```

## Show outputs

```bash
just output _template
```

## Clean up

```bash
just destroy-apply _template
```
